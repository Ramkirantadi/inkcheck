About the Project

InkCheck — AI Verification

A final-year graduation project using convolutional neural networks to detect genuine and forged handwritten signatures with high accuracy.

What is InkCheck?

InkCheck is an intelligent web-based signature verification system that uses a trained Convolutional Neural Network (CNN) model to classify uploaded signature images as either Genuine or Forged.

The system is built with Flask for the backend, TensorFlow/Keras for the deep learning model, and OpenCV for image preprocessing — wrapped in a modern, professional frontend interface.

How the CNN Model Works

1
Image Upload & Preprocessing
The uploaded signature image is read using OpenCV, converted to RGB colour space, resized to 64×64 pixels, and normalized to a 0–1 floating point scale to match the exact training pipeline.
2
CNN Feature Extraction
The preprocessed image is passed through multiple convolutional and pooling layers that extract spatial features such as stroke curvature, pressure patterns, loop geometry, and ink distribution across the signature.
3
Classification & Confidence Scoring
The final dense layers output a single sigmoid probability score. Scores above 0.5 are classified as Genuine; scores at or below 0.5 are classified as Forgery. The confidence percentage is derived from this score.
